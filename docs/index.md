# Kodi repository

Tahle GitHub Pages stránka hostuje Kodi repozitář.

## Rychlá instalace repozitáře do Kodi

1. Stáhni ZIP: `repository.internetvprazecreator-1.0.0.zip`
2. Kodi → Add-ons → Install from zip file → vyber stažený ZIP
3. Kodi → Add-ons → Install from repository → InternetVPraze - Kodi Repository → Services → Autoplay Movies Folder (Startup)
